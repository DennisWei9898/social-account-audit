# Prompt · ③ 研究（派 AI 三路並行）

> 貼給 Claude。先貼上你的訪談答案（INTERVIEW.md）與帳號現況，再貼這段。
> Claude 能開 subagent/workflow（如 Claude Code）時效果最好；純對話也能一路一路跑。

```
你是我的健檢指揮官。以下是我的訪談答案與帳號現況：
【訪談答案】<貼上 INTERVIEW.md 你的回答>
【帳號現況】<貼上近 12–15 則貼文主題+互動、bio、最好/最差貼文>

請當指揮官，派三路研究（能並行就並行）。每一路都：只查證、不腦補、每條發現附來源 URL，查不到就明說「查不到」。用繁體中文，最後彙整成一份 RESEARCH.md。

【第一路 · 受眾】
研究我想打的受眾：他們是誰、平常在乎什麼、有什麼痛點、會為什麼付錢。找出「他們願意掏錢」的證據（例如同類主題已成功的付費產品/課程的銷售數字）。

【第二路 · 同業整體】
分析「這個領域整體怎麼做」——盤點這個題材/niche 的創作者類型、主力內容格式、常見變現方式，找出「什麼有效、什麼是紅海」。
重點：看整片森林，歸納通則；**不要只盯一個帳號逐字抄**（太針對、也學不到全貌）。

【第三路 · 平台/變現】
研究適合我目標的引流與變現平台（若目標含開課：比較開課平台的抽成、上架門檻、客單價、適配性；數字附來源、無來源標「推估」）。

【交叉比對】
三路回來後，幫我找出一個「沒人好好佔、又剛好是我強項」的定位縫隙——這會是我的定位方向。若跟我的真實資產衝突就告訴我（別把市場空缺硬套成我不是的樣子）。
```

## 小抄
- 有 Claude Code：可要求它「開 subagent 並行三路、每路回報結構化 JSON、最後 synthesis」。
- 純對話：一路一路貼，最後叫它「把三路彙整、找定位縫隙」。
- 研究品質關鍵：**逼它附來源**。沒來源的「發現」就是它在猜。
